# React BabylonJS Real Time Diamond Render

## Libraries Used (Main)

- React JS
- Babylon Js
- Babylon Loader

## Babylon JS

- PBR Material 
  - Refraction
  - Reflection
  - LightMap Shader
  - envMap
  - Glb Loader from Babylon Loader


### `npm start`

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
